À propos de cette application

L'application "Administration Collègue" à été créée dans le cadre d'une formation de développeurs Java fullstack dans un contexte DevOps chez DTA Ingénierie. La création de cette application permet d'apprendre à créer une application Angular avec une partie backend développé dans un context Spring en simulant un système de gestion des collègues.
Vous trouverez ici le lien vers le dépot GitHub de l'application "Administration Collègues" et sa partie backend.
